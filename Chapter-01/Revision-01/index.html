Basics

HTML Essentials: The Skeleton

(WHAT) is it?

The structure of your web page. We use Tags like <form>, <input>, and <button> to build the user interface.

(WHY) do we use it?

Without HTML, the browser wouldn't know where to put the "Student Name" box or the "Submit" button. It's the layout of our camp!

(HOW) did we do it?

We used the name="..." attribute on inputs so PHP can identify them later. <input type="text" name="student_name">
Basics

PHP Essentials: The Brain

(WHAT) is it?

A server-side language that thinks for us. It starts with <?php and ends with ?>.

(WHY) do we use it?

HTML is just a static picture. PHP makes it "alive" by calculating grades, total marks, and averages based on what the user typed.

(HOW) did we do it?

We used Variables (starting with $) to store data like $name = $_POST['name'];. <?php echo "Hello, " . $name; ?>
Lesson 01

HTML Forms & Basic PHP

(WHAT) is it?

The very first step: Creating a UI for users to enter a student name and their mark, then processing it to find a Grade (A, B, C...).

(WHY) do we use it?

To learn how the web communicates. Every site (Facebook, Amazon) uses forms to take data and PHP (or similar) to make decisions.

(HOW) did we do it?

Used `$_POST` to catch data and `if-else` for decision making. <?php
if ($mark >= 75) { echo "Grade A"; }
?>
Lesson 02

Loops & Multiple Subjects

(WHAT) is it?

Scaling up to handle multiple subjects for ONE student.

(WHY) do we use it?

Because writing separate variables for "Math", "Science", and "History" is slow. Arrays let us group data together.

(HOW) did we do it?

Used `marks[]` in HTML to send an array, then used `foreach` in PHP to calculate the total and average. foreach ($marks as $val) { $total += $val; }
Lesson 03

The Full Classroom

(WHAT) is it?

Processing many students at once using dynamic forms.

(WHY) do we use it?

Real-world apps handle "collections" of data (a class of students, a cart of products).

(HOW) did we do it?

Used JavaScript `appendChild()` to add student blocks and Nested Loops to scan through each student's marks.
Lesson 04

Modular Functions

(WHAT) is it?

Moving logic into a special file called `functions.php`.

(WHY) do we use it?

Organization. It keeps your `index.html` clean and lets you reuse the same calculation in multiple projects.

(HOW) did we do it?

Used `include 'functions.php';` and wrote custom functions like `getLetterGrade()`. function getGrade($m) { return ($m >= 75) ? 'A' : 'B'; }
Lesson 05

Ranking & Sorting

(WHAT) is it?

Sorting the classroom results to find #1, #2, and #3.

(WHY) do we use it?

To turn data into **Information**. Knowing someone got an 80 is good; knowing they are the #1 in class is better!

(HOW) did we do it?

Used `usort()` to sort our student array by their Average score before displaying the table. usort($students, function($a, $b) { return $b['avg'] <=> $a['avg']; });
